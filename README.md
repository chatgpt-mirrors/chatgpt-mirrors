## 什么是 ChatGPT 镜像站？

ChatGPT镜像网站是指将原始网站的内容复制并放置在另一服务器上的网站。这个概念通常应用于提供备用访问途径，为主站点的繁重流量提供缓解。一般来说，镜像网站会更新以保持与原始网站相同的内容。
